# ISG Risk Generator V2

ISG (İş Sağlığı ve Güvenliği) Risk Değerlendirme Asistanı - Geliştirme Versiyonu

## Özellikler

- Google Gemini AI ile otomatik risk değerlendirmesi
- Fine-Kinney metodolojisi
- 50-200 arası risk oluşturma
- Excel çıktısı
- Türkçe destek

## Kurulum

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
